Two-Dimensional Shapes

2D shapes have two dimensions: length and width. Common examples include:

  1. Square

    No.Of Sides Perimeter Area
    4 4 x Side Side²

  2. Rectangle

    No.Of Sides Perimeter Area
    4 2 x (L + B) L x B

  3. Triangle

    No.Of Sides Perimeter Area
    3 3 x Side ½ (b × h)

  4. Circle

    No.Of Sides Perimeter Area
    0 2 x π x r π x r²
